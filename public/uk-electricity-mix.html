What is the UK electricity mix right now?

The current UK electricity mix is best checked on the live Energy Mix dashboard, which shows generation by fuel type, demand, carbon intensity, renewable share, low-carbon share and interconnector flows with source freshness indicators.

What this page is for

Use this page as a crawlable reference for the UK electricity mix. For live numbers, use the main dashboard at https://energymix.info/.

What counts in the mix

The dashboard includes major generation categories such as wind, solar, gas, nuclear, biomass, hydro, storage and other source categories where present in upstream data. It also shows imports and exports separately so users can understand net interconnector flows.

Why source freshness matters

Different electricity datasets update on different cadences. Fast generation feeds can update more often than settlement-period data, so Energy Mix displays freshness metadata to separate source timestamp from page refresh time.

Data sources and methodology

Energy Mix is an explanatory dashboard built on authoritative open electricity datasets, including Elexon BMRS/FUELINST, NESO sources, the Carbon Intensity API and interconnector context data where applicable. Read the methodology at /data.
